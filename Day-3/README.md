# Day 03 – Flexbox Layout & Navigation

On **Day 3** of the **50 Days Dev Web Development Challenge**, I explored **CSS Flexbox**, a modern layout system used to build responsive and well-structured web pages. I transformed the navigation bar into a flexible layout, positioning the community logo on the left and the navigation links on the right using Flexbox properties.

I also centered the Hero section both horizontally and vertically by applying Flexbox with a column layout. To improve the user experience, I completed the bonus challenge by styling the navigation links, removing the default underlines, and adding a hover effect that changes the link color using CSS variables.

### What I Learned 

* Understanding the Flexbox layout model
* Using `display: flex` to create flexible layouts
* Aligning elements with `justify-content` and `align-items`
* Changing layout direction using `flex-direction`
* Creating consistent spacing with `gap`
* Building a centered Hero section using Flexbox
* Styling navigation links and adding interactive hover effects with the `:hover` pseudo-class

### Technologies Used

* HTML5
* CSS3 (Flexbox)

### Outcome

Successfully built a responsive navigation bar and a centered Hero section using Flexbox while improving the website's appearance and interactivity with clean navigation styling and hover effects. This strengthened my understanding of modern CSS layout techniques and responsive web design fundamentals.
