# Day 01 – Semantic HTML Architecture

Today marks the beginning of my **50 Days Dev Web Development Challenge**. The focus was on building a strong foundation using **HTML5 semantic elements** without applying any CSS or JavaScript.

In this project, I created the basic structure of a community landing page for **Synexus Official** using semantic tags such as `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`. The page includes a navigation bar, hero section, about section, initiatives section, core team section, and footer while following accessibility and SEO best practices.

### What I Learned

* HTML5 semantic elements and their purpose
* Proper webpage structure and document hierarchy
* Basic SEO using meta tags
* Accessibility with ARIA roles
* Creating clean, meaningful, and maintainable HTML without unnecessary `<div>` elements

### Technologies Used

* HTML5

### Outcome

Successfully built an accessible, SEO-friendly, and well-structured webpage that serves as the foundation for future styling with CSS and interactivity using JavaScript.
