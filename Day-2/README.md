# Day 02 – CSS Box Model & Reset

On **Day 2** of the **50 Days Dev Web Development Challenge**, I focused on building a strong CSS foundation by understanding the **CSS Box Model**, applying a **Universal Reset**, and creating reusable **CSS Variables**.

I removed the browser's default margins and padding using the universal (`*`) selector and applied `box-sizing: border-box` to ensure elements are sized more predictably. I also introduced CSS custom properties (`:root`) to define reusable color variables, making the stylesheet easier to maintain and scale. Finally, I styled the `body` with a clean, readable font and applied the text color using CSS variables.

### What I Learned

* The importance of resetting default browser styles
* How the CSS Box Model affects element sizing
* Using `box-sizing: border-box` for consistent layouts
* Creating reusable CSS Variables with `:root`
* Applying global typography and text styling

### Technologies Used

* HTML5
* CSS3

### Outcome

Established a clean, consistent, and maintainable CSS foundation that will support responsive layouts and more advanced styling in the upcoming days of the challenge.
